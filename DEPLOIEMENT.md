**https://juniorcherif2004-star.github.io/zou-bootcamp/**

## Étapes de déploiement :

### 1. Créer le dépôt GitHub
1. Allez sur https://github.com/new
2. Nom du dépôt : `zou-bootcamp`
3. Public (recommandé pour GitHub Pages)
4. Ne cochez pas "Add a README file"
5. Cliquez sur "Create repository"

### 2. Préparer le projet localement
```bash
# Ouvrir Git Bash dans le dossier du projet
cd "c:\Users\m0660\Downloads\zou final\ZOU BOOTCAMP"

# Initialiser Git
git init
git add .
git commit -m "Initial commit - ZOU Bootcamp Site"

# Connecter à votre dépôt
git branch -M main
git remote add origin https://github.com/juniorcherif2004-star/zou-bootcamp.git
git push -u origin main
```

### 3. Activer GitHub Pages
1. Allez sur : https://github.com/juniorcherif2004-star/zou-bootcamp/settings/pages
2. Source : "Deploy from a branch"
3. Branch : "main"
4. Folder : "/ (root)"
5. Cliquez sur "Save"

### 4. Attendre le déploiement
- Patientez 2-5 minutes
- Votre site sera disponible à : https://juniorcherif2004-star.github.io/zou-bootcamp/
